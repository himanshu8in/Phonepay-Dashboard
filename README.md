# Payment Insights Dashboard

## Overview

This Power BI dashboard analyzes payment transactions and user activity to provide actionable business insights. The dashboard is designed for executives and stakeholders to monitor key performance indicators, transaction trends, user demographics, and platform performance.

## Key Metrics

* Total Transactions: 101K+
* Total Transaction Value: ₹528M+
* Active Users: 108K+
* Success Rate: 96.15%

## Features

* Interactive KPI Cards
* Transaction Trend Analysis
* User Demographics Breakdown
* Payment Success Monitoring
* Business Insights Panel
* Dynamic Filtering and Slicers

## Tools & Technologies

* Power BI
* DAX
* SQL
* Excel

## Business Insights

* Transaction success rate remains above industry benchmarks.
* User engagement continues to drive transaction growth.
* High-value users contribute significantly to transaction volume.
* Payment activity remains consistent across reporting periods.
* Strong platform adoption is reflected in active user growth.
